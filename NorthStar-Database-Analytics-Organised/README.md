# NorthStar Urban Mobility and Logistics Database and Analytics Project

This repository contains a bachelor-level database and analytics coursework solution for the NorthStar Urban Mobility and Logistics case study.

## Project Aim

The aim of this project is to analyse NorthStar's operational problems and propose a data-driven solution using SQL, R, Python, and MongoDB.

## Repository Structure

- `Python/` - Python scripts for cleaning, processing, analysis, and visualisation.
- `SQL/` - SQL queries for filtering, joining, aggregation, and optimisation.
- `R_Analytics/` - R scripts for statistical analysis and visualisation.
- `MongoDB/` - MongoDB database design, CRUD operations, aggregation, and indexing.
- `Notebook/` - Jupyter Notebook version of the full project.
- `Visualisations/` - Output charts and screenshots.
- `Report/` - Final coursework report or placeholder.
- `Data/` - Dataset folder or sample data placeholder.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- SQL / SQLite
- R
- ggplot2
- MongoDB
- PyMongo
- Jupyter Notebook

## Main Analytical Focus

- Customer complaints
- Failed deliveries
- Delivery delays
- Route performance
- Vehicle downtime
- Charging hub utilisation
- Query optimisation
- Hybrid SQL and NoSQL database design
